# 🌾 Smart Agriculture AI System

An intelligent, AI-driven platform designed to assist farmers and agriculture enthusiasts. This project provides **Crop Recommendations**, **Fertilizer Guidance**, and **Plant Disease Detection** using Machine Learning and Deep Learning.

---

### 🚀 Live Demo
**👉 [Launch Smart Agriculture App](https://smart-agriculture-ai-d2zgcaxfdnhx8wzjverisv.streamlit.app/)**

---

### ✨ Features
- **🌱 Crop Recommendation**: Suggests the optimal crop to cultivate based on soil and weather conditions.
- **🧪 Fertilizer Recommendation**: Advises on the best fertilizer for your soil type and crop.
- **🔍 Disease Detection**: Upload a picture of a plant leaf to instantly identify diseases and receive actionable treatment plans.
- **🤖 AI Chatbot**: An integrated intelligent assistant for all your agricultural queries.
- **🌤️ Local Weather Intelligence**: Incorporates real-time weather data for smarter agricultural insights.

### 💻 Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abhay-2003/smart-agriculture-ai.git
   cd smart-agriculture-ai
   ```

2. **Set up the virtual environment & install dependencies:**
   ```bash
   python -m venv venv

   # On Windows:
   venv\Scripts\activate
   # On Mac/Linux:
   source venv/bin/activate
   
   pip install -r requirements.txt
   ```

3. **Run the application:**
   ```bash
   streamlit run app/streamlit_app.py
   ```
   *The application will automatically launch in your browser at `http://localhost:8501`*
